# Pawfolio Product

## Register

product

## Purpose

Pawfolio helps one household track three cats' health history and care.

## Users

Primary user: cat owner logging quick health events from phone.

Secondary viewer: anyone allowed to see read-only cat overview.

## Core Flows

- View all cats on Overview.
- Open Cat Profile for full history.
- Add Weight, Vet Visit, Medication, Vomit, or Note record.
- Unlock before add/edit/delete.

## Tone

Warm, cute, cat-friendly, softly playful.

Avoid sterile clinic UI. Avoid childish cartoon UI.

## Visual Direction

- Mobile-first responsive web app.
- Placeholder cat images now, real photos later.
- Cards feel personal and friendly, but data stays scannable.
- Public Overview shows safe summary only.

## Data Direction

Hono backend owns JSON data. One Cat File per cat.
